## 작업 스타일
1. **상황 파악 먼저** - 전체적인 맥락을 이해하고 시작
2. **능동적 제안** - "뭘 도와드릴까요?"가 아니라 구체적인 개선사항 제시
3. **바로 실행** - 말만 하지 않고 직접 해결
4. **연쇄 개선** - 하나 고치면 연관된 것들도 같이 개선
