# Test Agent

능동적으로 문제를 찾아 해결하는 개발 도우미 에이전트

## 🚀 특징

- **능동적 문제 해결**: 요청을 기다리지 않고 스스로 개선점을 찾아 제안
- **자연스러운 대화**: AI스럽지 않은, 동료 같은 편안한 소통
- **즉시 실행**: 말만 하지 않고 직접 문제를 해결

## 📋 주요 기능

### 코드 & 개발
- 코드 분석 및 개선점 발견
- 버그 탐지 및 수정 제안
- 리팩토링 아이디어 제시
- 자동 문서화

### 파일 관리  
- 프로젝트 구조 최적화
- 불필요한 파일 정리
- 설정 파일 개선
- 문서 품질 향상

### 문제 해결
- 에러 로그 분석
- 성능 병목 지점 탐지
- 보안 취약점 점검
- 의존성 문제 해결

## 🛠️ 설정 파일

- `agent.json`: 에이전트 기본 설정 (Slack 연동, 모델 등)
- `agent.md`: 상세 기능 명세서  
- `AGENTS.md`: 시스템 프롬프트 (핵심 지침)

## 💬 사용 방법

그냥 자연스럽게 대화하면 돼! 상황을 파악해서 필요한 도움을 알아서 제공할게.

## 🚀 설정 및 관리

### 초기 설정
```bash
./setup.sh  # 개발환경 자동 설정
```

### 관리 명령어
```bash
./agent-utils.sh status        # 현재 상태 확인
./agent-utils.sh validate      # 설정 파일 검증
./agent-utils.sh monitor       # 모니터링 리포트
./agent-utils.sh monitor watch # 실시간 모니터링  
./agent-utils.sh backup        # 백업 생성
```

### 모니터링
```bash
python3 monitor.py        # 단일 상태 체크
python3 monitor.py log    # 로그 파일 저장
python3 monitor.py watch  # 지속적 모니터링
```

## 🛠️ 자동화 기능

- **Pre-commit 훅**: 커밋 전 자동 검증 (JSON, Python 문법 체크)
- **상태 모니터링**: 실시간 에이전트 상태 추적
- **자동 백업**: 설정 파일 자동 백업 시스템
